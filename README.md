# pharos-game-admission
My submission for the admission test for the Pharos Game project internship program. Being made in the Unity Engine, version 2019.2.1.

Keyboard Controls:  
Arrows - movement   
Z - melee attack  
X - ranged attack  
Shift - Dash  
  
Gamepad controls:  
Left analog stick - movement  
A - Jump  
X - melee attack  
Y - dash  
B - Ranged attack
